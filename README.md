# ArshGroup Website

React + Tailwind + Framer Motion project for ArshGroup (interior and construction company).

## Run

```bash
npm install
npm run dev
```

## Structure

- `src/components` reusable UI components
- `src/pages` route-level pages
- `src/data/siteData.js` content model
- `skills/frontend-design/SKILL.md` imported skill reference
